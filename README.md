# Lista de Tarefas Diária 

Projeto desenvolvido com HTML, CSS e JavaScript.

## Função principal
Ajudar na organização de tarefas diárias através de notificações e alarmes para que um possível cronograma seja executado de maneira mais eficaz. 

## Funcionalidades
- Adicionar tarefas com horário
- Alarme sonoro no horário definido
- Salvamento automático no navegador

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Como executar
Abra o arquivo `index.html` no navegador.
